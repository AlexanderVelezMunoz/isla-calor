Metodología IDW

El sistema utiliza interpolación espacial Inverse Distance Weighting (IDW) para representar el comportamiento térmico superficial urbano.

Integración SIATA

El sistema consulta información meteorológica mediante DOI de estaciones SIATA, permitiendo complementar los datos históricos con observaciones recientes para mejorar el análisis climático.

Sistema SIG Web

Plataforma orientada al análisis climático urbano para apoyar procesos de monitoreo, evaluación ambiental y toma de decisiones.
