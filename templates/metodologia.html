Metodología del sistema

El sistema SIG de isla de calor urbana se desarrolla bajo un enfoque híbrido que integra datos históricos, datos en tiempo real y análisis espacial.

1. Recolección de datos

  • Base histórica 2013–2024
  • API SIATA (datos meteorológicos en tiempo real)
  • Fuentes complementarias IDEAM

2. Procesamiento

Se utilizan librerías como Pandas y NumPy para limpieza, normalización y estructuración de datos climáticos.

3. Interpolación espacial (IDW)

Se aplica el método Inverse Distance Weighting para estimar la distribución espacial de la temperatura superficial.

4. Visualización

Los resultados se representan en mapas generados dentro del sistema SIG web.
